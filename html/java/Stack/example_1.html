有的时候，我们需要查看栈内的元素及其顺序分布。正在因为存在这个需求，我特意整理出来。

方法一：通过删除栈顶元素的方法来实现遍历，目前没有好的方法，2020年2月15日记

Stack <integer> stack =new Stack<integer>();
	for(int i=0;i<6;i++){
		stack.push(i);
	}
	for(int i=0;i<stack.size();i++){
		top=stack.peek();
		System.out.println(top);//获取栈顶元素
		stack.pop();//删除栈顶元素
	}
